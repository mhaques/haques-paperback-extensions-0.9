Haques Paperback Extensions

Custom manga and comic extensions for Paperback

Installation

Click to automatically add this repository to Paperback

OR

Manual Installation

  1. 1 Open Paperback app on your device
  2. 2 Go to Settings → External Sources
  3. 3 Tap Add Repository
  4. 4 Copy and paste the URL below:
https://mhaques.github.io/haques-paperback-extensions-0.9/0.9/stable/
Copied to clipboard!
  1. 5 Tap Add to add the repository
  2. 6 Install your desired extensions from the list
  3. 7 Start reading!

About

This is a custom fork of the Kakarot Extension repository, maintained by Haques. It includes MangaBuddy and Kaynscan sources for the Paperback reader app.

Note: These extensions are not affiliated with the source websites.

Requirements

  • Paperback 0.9.x or higher
  • iOS 15.0 or higher
  • Active internet connection
